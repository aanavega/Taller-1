 ## Cuarto punto
 Realice un programa que lea dos números reales y determine si el primero es múltiplo del segundo.
 
 - Explicacion:
   1) Definimos las variables, tres numeros reales (float).
   2) Hacemos un condicional para ver si el residuo del primer numero entre el segundo es 0.
   3) Si es 0, si es multiplo, si no, no es multiplo del segundo numero
